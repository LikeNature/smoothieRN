# reactNativeApp

## create-react-native-app 설치하고 사용하기

```bash
npm install -g create-react-native-app
```

## 리액트 네이티브 앱 실행하기

```bash
create-react-native-app testProject
```

## 앱 실행하기

```bash
npm start
```

## Expo 설치하고 사용하기

```bash
npm install -g expo-cli
```

```bash
expo init AwesomeProject

cd AwesomeProject
npm start # you can also use: expo start
```
